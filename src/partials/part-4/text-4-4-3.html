Программное обеспечение

Чтобы человек мог использовать компьютер для решения задач, на компьютер нужно установить программное обеспечение (англ. software).

Программное обеспечение (ПО) — это весь набор программ компьютера для ввода, обработки, вывода и передачи данных.

Всё программное обеспечение компьютера можно разделить на три группы — системные программы, системы программирования и прикладные программы.

Всех людей, кто работает с компьютерами, также можно разделить на три группы — системные администраторы, программисты и пользователи.

Пользователи используют прикладные программы, чтобы решать свои задачи. Примеры прикладных программ — это редакторы текстовой, графической, звуковой, видеоинформации, электронные таблицы, системы управления базами данных, CAD-системы, браузеры, игры и другие.

Программисты используют системы программирования, чтобы писать новые программы. Примеры систем программирования — Visual Studio Code, IntelliJ IDEA, Eclipse, PyCharm, Microsoft Visual Studio, Xcode и другие.

Системные администраторы настраивают системное программное обеспечение и все другие программы так, чтобы все, кто использует компьютер, могли решать свои задачи. Системное программное обеспечение — это комплекс программ. Задачи системных программ обеспечить:

  • работу всех устройств компьютера,
  • удобный интерфейс (способ обмена данными) между пользователем и прикладными программами,
  • удобный интерфейс между пользователем и устройствами компьютера.

В состав системного программного обеспечения входят операционные системы (например, Windows, Unix, Linux, macOS, Android, iOS и другие), драйверы (программы, которые управляют устройствами компьютера), утилиты (служебные программы для проверки и настройки компьютера).

Файловая система

Жёсткий диск — это устройство внешней постоянной памяти компьютера. Современные жёсткие диски могут иметь объём памяти больше 10 Тбайт. Большие жёсткие диски можно делить на логические диски (разделы).

Все устройства внешней памяти, которые подключаются к компьютеру (физические и логические диски), имеют логическое имя. Для операционной системы Windows логическое имя устройств внешней памяти — это буква латинского алфавита и двоеточие. Например, c:, d:, e: и так далее.

Часть операционной системы, которая определяет правила организации, именования, хранения информации на внешних носителях, — это файловая система. Основные объекты файловой системы — это файл и каталог.

Вся информация в памяти компьютера — программы и данные — хранится в виде файлов. Файл — это именованная область внешней памяти. Файл — это информация, которая имеет имя и хранится во внешней памяти компьютера как один объект.

Каталог — это именованный список файлов и других каталогов. Каталоги нужны, чтобы разделить файлы на группы для удобной работы.

Основные операции с файлами и каталогами, которые выполняет файловая система, — это создание, переименование, копирование, перемещение и удаление.

Логическое имя устройства внешней памяти — это имя корневого каталога. Все другие каталоги, которые находятся на внешнем носителе внутри корневого каталога, — это вложенные каталоги. Все каталоги, начиная с корневого, образуют дерево каталогов.

Каталоги и файлы имеют имя. Имя любого вложенного каталога — набор символов, слово или фраза — это информация о том, что находится в этом каталоге.

Полное имя файла — это набор символов, который состоит из двух частей — короткого имени и расширения. Короткое имя файлу может давать пользователь, который создаёт файл, или программа, в которой пользователь сохраняет файл. Короткое имя файла — это информация для пользователя о том, что находится в данном файле. Расширение файлу даёт программа, в которой пользователь сохраняет файл. Расширение — это информация для пользователя и для операционной системы. Пользователь по расширению может определить, какой вид информации хранится в файле — текст, рисунок, музыка, программа и так далее. Операционная система по расширению определяет, какую программу нужно использовать для работы с данным файлом. В таблице 1 приведены примеры типов файлов и примеры их расширений.
