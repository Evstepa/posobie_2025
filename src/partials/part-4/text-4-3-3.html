Количество информации

Измерение информации — это задача, которую изучает теоретическая информатика.

Компьютер работает с данными. При работе с данными важно знать, сколько памяти нужно иметь, чтобы сохранить данные, сколько времени нужно, чтобы передать данные от одного компьютера другому компьютеру. Для этого можно определить количество знаков, которыми записаны данные. Это объёмный подход к определению количества информации.

Количество информации в сообщении равно количеству знаков в этом сообщении. Значит, разные сообщения с одинаковым количеством знаков будут иметь одинаковый объём информации.

Чтобы записать данные в виде, удобном для обработки на компьютере, используют цифры 0 и 1. Цифры 0 и 1 — это двоичные цифры (англ. binary digits).

Бит (англ. binary digit) — это количество информации, которую можно записать с помощью одной двоичной цифры. Бит — это наименьшая единица количества информации.

1 бит — это количество информации, в котором могут быть только два различных варианта. Если вариантов больше, то битов для записи сообщения должно быть больше. Значит, информации в сообщении будет больше. Количество вариантов N и количество битов информации I связаны формулой N = 2I. Из этой формулы получим: I = log2 N. Это формула Хартли.

С помощью формулы Хартли можно определить теоретическое количество информации в сообщении, которое содержит L символов: I = L · log2 N. Это алфавитный подход.

Байт (англ. byte) — это основная единица количества информации. 1 байт = 8 бит.

Кодирование информации

Чтобы информацию было удобно обрабатывать на компьютере, информацию нужно закодировать.

Кодирование — это запись информации с помощью знаков в форме, удобной для хранения, обработки и передачи. Правило, по которому выполняют кодирование информации, — это код. Декодирование — это обратный переход от закодированного сообщения к исходной информации.

Коды можно разделить на равномерные и неравномерные. При неравномерном кодировании коды разных символов могут быть разной длины (например, код Морзе). При равномерном кодировании коды всех символов алфавита имеют одинаковую длину.

Двоичное кодирование — это запись информации с помощью цифр 0 и 1.

В конце XVII века немецкий учёный Г. В. Лейбниц создал механическую вычислительную машину, в которой использовал кодирование чисел с помощью цифр 0 и 1.

Для кодирования информации, представленной в разных формах, используют разные правила.

Для кодирования числовой информации используют запись чисел в двоичной системе счисления. Для дополнительной информации прочитайте текст «Двоичное кодирование числовой информации».

Для кодирования текстовой информации используют специальные стандарты — кодовые таблицы. Основными являются стандарт ASCII с расширениями (ANSI) и стандарт UNICODE. Для дополнительной информации прочитайте текст «Кодовые таблицы ASCII, ANSI и UNICODE».

При кодировании графической информации используют два метода — растровое кодирование и векторное кодирование. Для дополнительной информации прочитайте тексты «Растровое кодирование» и «Векторное кодирование».

Для кодирования звуковой информации используют устройства, которые преобразуют звук в электрический сигнал (микрофон), а затем электрический сигнал преобразуют в двоичный код (звуковая карта).

Чтобы закодировать видеоинформацию, нужно закодировать звук, закодировать изображение, которое изменяется, и соединить их так, чтобы звук и изображение совпадали.
